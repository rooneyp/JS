Whitelist Web Sites

Blacklist Web Sites

Use blacklist mode. (Pop ups allowed by default. Refresh web pages to take effect when changing this option.)

To whitelist, enter the address of the web site (one per line). For example, http//www.hulu.com whitelists Hulu by searching for the above string at the beginning of a URL.

Alternatively, you can use regular expressions if you start the expression off with a caret. For example, ^(http[s]?:\/\/[a-z0-9\._-]+\.|http[s]?:\/\/)google\.[a-z]+($|\/) is the regular expression to whitelist all Google web sites regardless if they are http or https and regardless of whether its .com, .ca, .info, etc.... (except for double dot domains like .co.uk) 

All URLs and regular expressions are case insensitive.
